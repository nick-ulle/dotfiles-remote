# Remote Computing Dotfiles

These are the dotfiles @nick-ulle uses to set up on remote computing resources.


## Usage

This repository depends on [Pixi][] to install dotfiles and global packages.

[Pixi]: https://pixi.sh/

To install:
```bash
pixi run install
```

To reinstall (after edits):
```bash
pixi run reinstall
```

To uninstall:
```bash
pixi run uninstall
```
